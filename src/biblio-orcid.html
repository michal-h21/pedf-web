Zavádění personálních identifikátorů ORCID na PedF UK

V průběhu roku 2017 je na základě Opatření rektora č. 28/2016 na UK zaváděno užívání personálních identifikátorů, především indentifikátoru ORCID (Open Researcher and Contributor ID).

Toto Opatření ukládá autorům:

  • využívat personální identifikátory, tj. je zřídit a aktivně používat  

  • povinnost zřídit si ResearcherID (jsou-li jejich publikace evidovány v databázi Web of Science)  

  • zřídit si ID ORCID  

  • propojit si personální ID ORCID s ResearcherID a SCOPUS Author ID (je zřízen automaticky při 1.publikaci autora, která je evidována v databázi Scopus)  

  • doplnit do ORCID své publikace (v blízké budoucnosti bude proveden hromadně import záznamů z OBD)  

  • pravidelně aktualizovat ORCID ID profil (včetně aktualizace přiřazení nových publikací přes ResearcherID a Scopus Author ID)  

  • uvádět ORCID ID u všech publikací  

  • uvádět u všech ID a všech publikaci anglickou variantu afiliace dle přílohy č. 2 OR 18/2012, více viz Správná forma uvádění institucionální afiliace autorů z PedF UK –  

 

Za autory jsou zde považováni: akademičtí a vědečtí pracovníci, doktorandi, další publikující zaměstnanci a studenti

 

Pracovníci nově přijatí mají povinnost zřídit si ORCID ID do 31.10.2017, další noví členové Akademické obce nahlásí svému katedernímu správci do 2 měsíců od nástupu či přijetí ke studiu své ORCID ID nebo si jej vytvoří a nahlásí ho svému katedernímu správci

ID jsou shromažďována (prostřednictvím katederních správců OBD) v tabulce , která bude sloužit pro propojení záznamů publikační činnosti v OBD s ID ORCID.

Návody pro zřízení ID ORCID a propojení s dalšími ID jsou k dispozici na stránkách Ústřední knihovny UK

Pracovní skupina pro metodickou pomoc při zavádění identifikátorů PedF: Mgr. Jitka Bílková, Martina Růžičková, Blanka Vorlíčková (ÚK PedF)

Vytvořená ID ORCID budou implementována do Whois NT, v budoucnu bude OBD napojena na databáze WOS a SCOPUS

Identifikační čísla ORCID z tabulky budou během září 2017 doplněna do Whois.

Další informace viz Prezentace k zavádění personálních identifikátorů na UK ze setkání katederních správců PedF dne 6. února 2017.
